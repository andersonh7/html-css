q e blockquote

Esse é bem simples o <q> serve para fazer isso Botar as coisas entre aspas. É bom para ser usado para falas, ainda mais que o HTML5 é feito para ter significado
O blockquote é um pouco mais complexo/completo. Já que ele faz isso

coloca o texto separado, da pra usar como fala também eu acho. Mas a principal função é que da pra colocar tipo a "fonte" de tal dado nisso aqui. Não vai interferir na experiêscia do usuario, mas ajuda no mecanismo de busca do Google e essas coisas. Tipo nesse texto mesmo, tem um "cite" que é usado tipo uma "class" e é ai que coloca a URL da fonte. No cite esta a exata aula em que foi ensinado isso
Esse aqui é só um texto para ver como funciona o espacamento do blockquote. Realmente põe um espaço!

abbr

Isso aqui é bem util. A tag <abbr> serve para colocar uma descrição em pop-up em uma determinada palavra. Quer ver um exemplo? Passe o mouse sobre a tag abbr qualquer uma das duas, etm descrição nas duas! Da pra ver que tem a descrição por ter esse traço embaixo (muda de acordo com o navegador, de acordo com a aula), mas isso aparentemente é editavel usando o css
